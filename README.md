# School Mosque Management System

A modern, scalable web-based platform for managing mosque and school activities, spiritual resources, educational programs, and community engagement.

## Tech Stack

- **Frontend**: React.js + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express.js + TypeScript
- **Database**: PostgreSQL
- **Authentication**: JWT
- **State Management**: Redux Toolkit

## Getting Started

### Prerequisites

- Node.js (v18+)
- PostgreSQL (v14+)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies for both client and server
3. Configure environment variables
4. Run database migrations
5. Start development servers

See detailed setup instructions in `/client/README.md` and `/server/README.md`
